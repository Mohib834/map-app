
How to the update button icons.

svg icons can be edited then converted into dataurls for use in class css.
After saving out from illustrator, open the svg in a text editor and remove the width and height values, otherwise the ones in the css won't be appled
convert them to dataurls (urlencoded) using a website and replace in css.
